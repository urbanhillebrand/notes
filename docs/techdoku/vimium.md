# vimium

**📌 NOTE**\
Mit `?` Keybinds anzeigen!

**Adressen öffnen**

|     |     |
| --- | --- |
| o | open URL/bookmark/history entry in current tab |
| O | open URL/bookmark/history entry in new tab |
| b | open bookmark |
| B | open bookmark in new tab |
| f | Link auf Seite öffnen |
| F | Link auf Seite in neuem Tab öffnen |

**Tabs**

|     |     |
| --- | --- |
| t | neues Tab |
| K | tab right |
| J | tab left |
| x | Tab schließen |
| X | Tab reopen |
| g0 | erster Tab |
| ngx | gehe zu Tab +x |
| yt | duplicate current tab |
| T | geöffnete Tabs durchsuchen |
| W | akt. Tab in neuem Fenster öffnen |
| &lt;a-p> | pin/unpin current tab? |

**History**

|     |     |
| --- | --- |
| H | back |
| L | forward |

**Scrolling**

|     |     |
| --- | --- |
| u / d | half page up / down |
| &lt;c-f> | full page down |
| &lt;c-b> | full page upKKKG |

**Sonstiges**

|     |     |
| --- | --- |
| r | reload |
| / | search |
| n / N | jump next/previous match |
| yy | yank URL |
| yf | yank Link |
| i | insert mode |
| gi | springe in erste Input-Box auf Seite |
| &lt;n> | Wiederholung des commands, z.B. "5t" öffnet 5 Tabs |

**Marks**

|     |     |
| --- | --- |
| ma, mA | set mark / global mark "a" |
| `a, `A | ump to mark / global mark "a" |
| `` | zurück zu letzter Position springen |

**URL Bearbeitung**

|     |     |
| --- | --- |
| gu | up one level in URL hierarchy |
| gU | up to root level in URL hierarchy |
| ge | edit current URL |
| gE | edit current URL and open in new tab |
