# Chrome

* Datenverzeichnis ermitteln: `chrome://version`

Bei Problemen mit fehlerhaften permanenten 301-Redirects

* Developer Tools, Network, `Disable cache`, Seite neu laden, Cache wieder enablen
* oder: Dev Tools, Network, rechts auf Request klicken & `Clear browser cache`
* oder: Dev Tools, lange auf den Reload-Button klicken, `Empty Cache & hard reload`
