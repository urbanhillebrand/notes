## Shortcuts

**telescope**

* `<Leader>ff` fuzzy-find files
* `<Leader>fg` grep files

**transparent**

* `<Leader>bg` toggle background transparency

**neo-tree**

* `<Leader>e` toggle neo-tree
* `a` add file/dir
* `d` delete file/dir

**lazygit**

* `<Leader>lg` display lazygit
