# zathura

PDF-Reader mit vim keyboard bindings

**Tipps**

* page fit: `a`
* fit width: `s`
* side by side: `d`
* rotate: `r`
* toggle white/black background: `ctrl-r`
* Links anzeigen: `f` (dann Linknummer + Enter zum springen)
* `F` gleich wie `f`, aber Link wird nur angezeigt
* mit `:offset <number>` kann der Offset am Buchbeginn manuell eingestellt werden (zieht nur für Sprünge mit `<number>G`, nicht jedoch für `:<number>`!)
