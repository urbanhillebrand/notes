# less

* **Diverses**
```
v # File in Editor öffnen!-
hjkl # vim-like Navigation
F # in Modus wie "tail -f " wechseln
ESC-F # wie F, bricht aber ab sobald das letzte Search-Pattern gefunden wird
ESC-u # undo search highlighting
:e <filename> # neues File aufmachen
= # Info anzeigen (Filename, Ort usw.)
gg # Beginn
G # Ende*
```

* **Shell-Commands**
```
! <shell-command> # führt den cmd aus - fügt man ein % ein, wird das aktuelle File übergeben (z.B. ! wc -l %)
```

* **Arbeiten mit mehreren Files**
```
:n # nächstes File aufmachen
:p # vorheriges File aufmachen
:x # erstes File aufmachen
```
