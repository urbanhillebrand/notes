# Markdown

## Textformatierung

|     |     |
| --- | --- |
| bold | ***text*** |
| italics | **italics** |

## Überschriften

|     |     |
| --- | --- |
| Überschrift 1 | # H1 |
| Überschrift 2 | ## H2 |
| Überschrift 3 | <mark>#</mark> H3 |

## Listen

unordered durch vorangestelltes `*` oder `-` oder `+`

ordered durch vorangestelltes `1.` (Ziffer kann konstant bleiben)

**nested mit Einrückungen:**

```
1. item1
1. item2
  1. subitem
1. item3
  * unordered subitem
```

## Code

Einzelne Zeilen / Wörter zwischen Backticks stellen

Mehrzeiligen Code zwischen "fences" mit stellen; z.B.

```
```
blah code
```
```

## Quotes

Blockquotes durch vorangestelltes `>`. Verschachtelte Quotes durch `>>` usw.
