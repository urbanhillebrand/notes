# LazyVim

## Keymaps

**general**

| key | desc |
| --- | --- |
| `<space>` | Leader |
| `\` | localleader |
| `C-h/j/k/l` | move between windows |
| `C-up/down/left/right` | resize windows |
| `A-j/k` | move down/up |
| `[b / ]b` | prev/next buffer |
| `esc` | escape & clear hlsearch |
| `<leader>l` | lazy |
| `<leader>fn` | new file |
| `<leader>qq` | quit all |

**Telescope**

| key | desc |
| --- | --- |
| `<leader><space>`, oder `<leader>ff` | Files im Projekt finden |
| `<leader>fr` | recent files |
| `<leader>` | recent files |

**search**

| key | desc |
| --- | --- |
| `<c-s>` | toggle flash search |
| `s` | flash search |
| `S` | flash treesitter? |
| `r` | remote flash |
| `R` | treesitter search |
| `<c-space>` | treesitter incremental selection |

**buffers**

| key | desc |
| --- | --- |
| `<leader>bb` | switch buffer |
| `<leader>bd` | delete buffer |
| `<leader>bD` | delete buffer and window |

**windows**

| key | desc |
| --- | --- |
| `<leader>-` | split below |
| `<leader>/` | split right |
| `<leader>wd` | delete window |
| `<leader>wm` | toggle zoom |

**tabs**

| key | desc |
| --- | --- |
| `<leader><tab><tab>` | new tab |
| `<leader><tab>` | next tab |
| `<leader><tab>f` | first tab |
| `<leader><tab>l` | last tab |
| `<leader><tab>d` | delete tab |
| `<leader><tab>o` | close other tabs |

**comments**
