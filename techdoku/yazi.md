# yazi

Navigation / Suche mit vim-Keys

**diverse Keybinds**

|     |     |
| --- | --- |
| . | versteckte Dateien anzeigen / verstecken |
| , | Sortierung |
| a | File oder Dir erstellen (Dir mit / am Ende) |
| r | rename |
| f | filter |
| M | gvfs-Mounts (gvfs-plugin) |
| s | Dateisuche mit fd |
| S | Dateiinhalte mit ripgrep durchsuchen |
| space | toggle selection |
| o oder Enter | open |
| O oder Shift-Enter | open interactively |
| d | delete |
| D | permanently delete |
| v | visual block mode |
| V | unset visual block mode |
| z | suche nach Dir/File via fzf (bei File wird dieses in der Preview angezeigt, nicht geöffnet) |
| Z | Verz. via zoxide suchen |
| K / J | in Preview-Pane 5 Zeilen rauf / runter |
| Ctrl a | alles selektieren |
| Ctrl r | Auswahl umkehren |
| Tab | show file info |

**Copy/Paste**

|     |     |
| --- | --- |
| y | yank |
| x | cut |
| p | paste |
| P | paste (overwrite) |
| Y or X | cancel yank status |

**Namen/Pfade kopieren**

|     |     |
| --- | --- |
| c c | file path |
| c d | directory path |
| c f | filename |
| c n | filename ohne extension |

**Tabs**

|     |     |
| --- | --- |
| t | new tab |
| 1 - 9 | change tab |
| Ctrl c | close current tab |
